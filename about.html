About

What is Fairytime Studio?

Fairytime studio is a hub for visual novels and adventure games inspired by retro anime.

I want to create games that feel magical and capture the vibes of turning on the TV in the late, late afternoon as the sun disappears behind the treeline. The sort of stories that convinced me that there were worlds beyond my little cul-de-sac, past the borders of my country, beyond the horizon or the broad expanse of space and even further so.

There are so many stories I want to tell and I hope you'll join me for the ride.

Releases

In order to push out my games as fast as possible I decided I'd like to try something different. Instead of completing one full game and releasing it, I'll instead release an episode at certain intervals through out the year. That way you can play the game as each episode is completed.

How often will you release episodes?

Once a month at first. Newer projects require more assets to be created before the game can be released. But as more assets are created I'm hoping the window between releases will increase to every two weeks.

How about choices and routes across episodes?

At the end of each episode you'll receive a code that you can save onto the website. When the next episode is on the website you'll be able to insert this code to get back onto your route.

What are slots?

Because the website I'm using has limited storage I can only post so much content so whenever a new episode is released I'll have to make room for it by removing a previous episode. This is where the slot system comes into play. There two types of slots:

New Slot - This is where you'll find the most recent episode and
Rerun Slot - This slot will have all the previous slots on a weekly rotation. For example; if there are three episodes of a game, the third episode will be in the New Slot while episodes one and two will cycle each week so you have an opportunity to play an episode you haven't yet played.

Okay. But what if there's a tonne of episodes?

A third slot will be put on the site. A Hyper Slot. This will cycle through all the episodes daily.

How many games are you planning to release?

Alot.
I have so many game ideas that I'm alittle scared I may not get to all of them in this lifetime. But I do have a roster of the ones I've deemed to be the best AKA the most interesting ones. Before starting a new project I'll probably put a poll up to see what y'all think the next project should be but for now there's three games on my priority list, including Glassheart.
